# 王78バス確認アプリ v2

Safariで使うための小型Webアプリです。

## v2の変更点

- 王７８のような全角表記を、王78として扱えるように修正
- 停留所名に補足表記が付く場合でも拾いやすいように修正
- Safari側の古いキャッシュが残りにくいようにファイルを更新

## 対象

- 都営バス 王78
- 南常盤台 → 王子五丁目
- 王子駅前方面
- 平日・土曜・休日の時刻表
- 指定時刻以降の3便
- 車両位置と停留所一覧

## 注意

- 遅れ時間は車両位置と予定時刻を使った概算です。
- まだ運行開始前の便、または便番号を車両位置と結び付けられない場合は「未判定」と表示します。
