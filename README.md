# 王78バス確認アプリ

Safariで使うための小型Webアプリです。

## 対象

- 都営バス 王78
- 南常盤台 → 王子五丁目
- 王子駅前方面
- 平日・土曜・休日の時刻表
- 指定時刻以降の3便
- 車両位置と停留所一覧

## 公開方法

このフォルダを、そのまま静的Webサイトとして公開します。

使いやすい方法は次のいずれかです。

1. GitHub Pages
2. Cloudflare Pages
3. Netlify

公開後、iPhoneのSafariでURLを開き、共有ボタンから「ホーム画面に追加」を選びます。

## 注意

- 公式の時刻表データと車両位置データを、ブラウザから直接取得します。
- 遅れ時間は車両位置と予定時刻を使った概算です。
- まだ運行開始前の便、または便番号を車両位置と結び付けられない場合は「未判定」と表示します。
- データ提供元がブラウザからの通信を制限した場合は、中継用の小さなサーバーが必要になります。
